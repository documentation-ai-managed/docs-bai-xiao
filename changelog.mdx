<Expandable title="Detailed Migration Guide for v1.2.0" default-open="false">

If upgrading from v1.1.0, update your webhook endpoints:

````javascript
// Before (v1.1.0)
fetch('https://api.baixiao.com/v1/webhooks', {
  headers: { Authorization: `Bearer ${YOUR_TOKEN}` }
});

// After (v1.2.0)
fetch('https://api.baixiao.com/v1.2/webhooks/events', {
  headers: { Authorization: `Bearer ${YOUR_TOKEN}` }
});
````

</Expandable>
