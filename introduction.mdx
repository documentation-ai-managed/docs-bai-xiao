## Overview

Bai Xiao provides a centralized platform where you organize and manage all your project documentation. Create spaces for teams, embed code examples, and collaborate seamlessly without scattered files. Start building your knowledge base today.

## Key Features

Bai Xiao offers powerful tools to streamline your documentation workflow.

<Columns cols={3}>
  <Card title="Organized Spaces" icon="folder" href="#spaces">
    Group docs by project or team in dedicated spaces with custom permissions.
  </Card>
  <Card title="Rich Editing" icon="edit-3" href="#editing">

## Benefits of Centralized Documentation

Using Bai Xiao eliminates version conflicts and lost files. You access everything from one dashboard, reducing onboarding time for new team members by up to 50%. Search across spaces instantly, and export to PDF or HTML for sharing.

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Visit `https://bai-xiao.com` and sign up with your email or GitHub account.
  </Step>
  <Step title="Set Up Space" icon="folder-plus">
    Click **New Space** and name it for your project, like "MyApp Docs".
  </Step>
  <Step title="Add First Doc" icon="file-plus">
    Create a new page and start writing. Use the editor for Markdown and components.
  </Step>
  <Step title="Publish" icon="globe">
    Save and share the public link or invite collaborators.
  </Step>
</Steps>

For API-driven setup, use this example to create a space programmatically:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/spaces', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
    body: JSON.stringify({ name: 'MyApp Docs' })
  });
  const space = await response.json();
  console.log(space.id);
  ```
  ```python
  import requests
  response = requests.post(
      'https://api.example.com/v1/spaces',
      headers={'Authorization': 'Bearer YOUR_API_KEY'},
      json={'name': 'MyApp Docs'}
  )
  space = response.json()
  print(space['id'])
  ```
</CodeGroup>
